# Chrome Web Store screenshots (1280×800)

Store-ready screenshots at the exact size the Chrome Web Store requires (**1280×800**). Downscaled
from the 2× originals in the parent folder; `popup.png` is the toolbar popup centred on a 1280×800
canvas.

Chrome allows **up to 5** screenshots — suggested set: `sky-3d`, `solar-system`, `panchang`,
`kundali`, `popup`. (Firefox/AMO accepts these too, or the 2× originals.)
